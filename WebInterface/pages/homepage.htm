Welcome to the "The Three Little Pigs" Siri Proxy!

Hello and welcome to the "The Three Little Pigs" Siri Proxy server.
This server must be fed with a 4S key in order to work, so please make sure to feed it or check the server status page.

At the moment this is a public Siri Proxy server and everyone can connect. When people stop donating keys, this might change.
So please, if you have an iPhone 4S, feed the piggy!
