The Three Little Pigs, how does it work?

I'm not going to explain everything with all the details etc but just very short and what is different about it in comparison with other Siri Proxies.

When someone donates a 4S key, it is stored in the database. Per key a small amount of users is allowed to protect the key from getting used too much and eventually getting blacklisted. When the server notices that there are a lot of requests on a specific key, the key gets hot. And when the key is getting too hot, it gets disabled to protect a ban from Apple. Although the key can be unblacklist very easily.

Other than that, this is just a Siri Proxy like antoher one but with several bug fixes.
But because of this genius system, developed by @JimmyKane9, this can be a public port as long as people keep donating keys.
